/ All sites

iSoftrepo is a web service providing information about installed software packages, which are managed by Advanced Module System (AMS). In AMS terminology, software packages are called modules. At every site, modules are grouped to thematic categories providing a fast way how to select software, which best matches your requirements. Every module is described in details. A short summary, license type, available versions, and links to software manual documentation are provided whenever possible.
